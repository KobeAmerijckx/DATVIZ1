Doormiddel van image processing vond ik de meest voorkomende kleurengroep op set intervals van de eerste vier
afleveringen van Boku no Hero: Academia. Deze kleurengroepen werden opnieuw geanaliseerd om de meest voorkomende kleur
te vinden. op deze manier krijgen we de 4 meest voorkomende kleuren in Anime.
